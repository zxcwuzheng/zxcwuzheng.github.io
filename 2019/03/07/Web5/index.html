Javascript的作用和特点

一般情况，我们可以将JavaScript看作是java语言的简化版本。
JavaScript可以分为三个部分：

  • 核心，是语言的内核，包括操作符，表达式，语句和子程序
  • 客户机端：应用的JavaScript是一组对象的集合，利用这些对象可以对浏览器和用户交互进行控制
  • 服务器端：同上，支持数据库管理系统之间的通信。

JavaScript的特点：

  • 分布式运算
  • 安全可靠
  • 容易移植
  • 使用浏览器对象

JavaScript语法

常量

1.整型常量
十六进制以0x或0X开头，例如：0x8a
八进制必须以0开头，例如：0123
十进制的第一位不能是0（数字0除外），例如：123

2.实型常量
12.32、193.98、5E7、4e5等。

3.布尔值
true和false

4.字符串和常量
“a book of JavaScript”、”abc”、”a”、””。

5.字符串中的特殊字符，需要以反斜杠()后跟一个普通字符来表示，例
如：\r(回车)、\n(换行)、\t(table键)、\b(退格键)、\’、\”、\。

注意:一个连续的字符串不能分开两行来编写，如果非要断开，就要分成两段字符串，中间用”+”来连接。

6.null常量

7.undefined常量
表示一个常量还没有进行赋值，或者是某个对象的属性不存在。
与null的区别，null常量有值，只不过是空的，但undefined常量压根就没值。

变量

JavaScript中的变量声明用关键字var引导：
声明原始类型的变量：
var 变量名 = 值；
声明引用类型的变量：
var 变量名 = new 类型名（初值）；
JavaScript的变量是弱类型的，可以随时更改它的类型。

类型

JavaScript的数据类型可分为两大类：

  • 原始类型：变量的值存放在栈中，包括Undefined，Null，Boolean，Number，String五种
  • 引用变量：栈中存放的是指针，其值另外放在堆中，包括Boolean，Number，string,Array，Date，Math等

浏览器对象模型BOM

在JavaScript中对象之间不是独立存在的，对象与对象之间有着层次关系。BOM就是用于描述这种对象与对象之间层次关系的模型，其中window对象是BOM的顶层对象，其它对象都是该对象的子对象。

window对象

window对象表示浏览器中打开的窗口，提供关于窗口状态的信息。
（1）调整窗口的大小和位置

  • window.moveBy：~（20，20）就是将窗口右移和下移20px
  • window.moveTo：~（20，20）是将窗口移动到（20，20）的位置
  • window.resizeBy：~（20，20）是将窗口宽度和高度增大20px
  • window.resizeTo：~（20，20）是将窗口宽度和高度设置为20px

（2）打开新窗口
用法如下：
window.open(url,target,options)
url：打开的浏览器窗口加载的URL
target：新打开的浏览器窗口的定位目标或者名称
option参数可选包括：

（3）系统对话框

  • window.alert：该方法将显示消息提示框
  • window.confirm：显示一个确认对话框
  • window.prompt：显示一个对话框，其中包含一个文本输入框

（4）状态栏控制

  • window.status：显示浏览器状态栏的信息

（5）定时操作

  • window.setInterval：用于设置定时器，每隔一段时间执行指定代码
  • window.clearInterval：清除setInterval函数设置的定时器
  • window.setTimeout：在一段时间之后执行指定代码
  • window.clearTimeout：清除setTimeout设置的定时器

document对象

document对象实际上window的属性

location对象

  • location对象中包含了当前窗口的url信息。
  • location.hostname 返回 web 主机的域名
  • location.pathname 返回当前页面的路径和文件名
  • location.port 返回 web 主机的端口 （80 或 443）
  • location.protocol 返回所使用的 web 协议（http:// 或 https://）

navigator对象

  • navigator对象保存浏览器厂家，版本和功能的信息
  • appCodeName 返回浏览器的代码名
  • appName 返回浏览器的名称
  • appVersion 返回浏览器的平台和版本信息
  • browserLanguage 返回当前浏览器的语言

screen对象

screen对象的属性保存最终用户的屏幕分辨率和屏幕绘制文档的功能信息

history对象

history对象保存当前对话中用户访问的url信息

利用Javascript可以对这些事件进行处理，创建动态页面。

在对HTMl文档进行动态修改时需要了解对象的位置，这里介绍使用元素名称进行元素定位。

获取表单中第一个输入框的定位格式如下：

获取表单中value=male的单选按钮及其属性定位格式如下

事件处理程序的引入

（1）静态引入
事件处理程序的句柄名以属性名的形式在HTML标记实体对象中出现，而对应的属性值为事件响应的代码，如下面这个例子：
<a onmouseover="showSomeMessage()" href="#">
（2）动态引入
将事件响应的函数名直接复制给HTML页面中某个标记元素的事件属性

常用事件

1.鼠标单击事件onclick
当操作者的鼠标指向某个HTML标记元素对象并单击鼠标时将触发onclick事件
<img src = "myimg.jpg" onclick="window.alert('图像被点击');">

2.onmousedown事件
当按下鼠标的某一个键时触发该事件

3.onmouseup事件
释放鼠标时触发改事件

4.onmouseover事件
当鼠标移动到一个对象上面时触发该事件。下面代码实现在鼠标移动到超链接上时在状态栏中显示“你好吗”。
<a href=http://www.163.com onmouseover="window.status=’你好吗‘； return true”>请点击</a>

5.onmouseout事件
当鼠标离开一个对象时触发该事件，这个事件适用于区域，层及超链接对象。

6.文本输入框及选择域的内容变化事件onchange
当表单文本域（如单行或者多行文本输入框）中的选择发送改变是触发事件，当在选择域（如下拉选择框）中选择某个选项时也会触发该事件，支持该事件的标记由<input type="text"> <select> <textarea>

7.选择事件onselect
当操作者在单行或多行文本输入框中对其中的字符选择时触发该事件

8.获得焦点事件onfocus
页面元素获得焦点时会触发该事件，如用户选中了单行或者多行文本输入框时

9.失去焦点事件onblur
如在文本框中输入结束并单击文本框以外的部分时触发该事件

10.页面文档的载入事件onload和卸载事件onunload
当浏览器加载完某个页面文档中的各个标记内容时触发该事件。只适用于`

11.图像事件（image events）
图形事件只能应用于<img>标记元素中，并且只提供了一个onabort事件，当某个图象文件加载中断时触发事件

12.键盘事件(keyboard events)
主要有onkeydown(按下)，onkeypress（按下后松开），onkeyup（松开）

Javascript表单处理

表单提交事件onsubmit
当用户单击表单中的提交按钮来提交表单中的各个请求时，就会触发表单的onsubmit事件。

select选择元素

checkbox复选框

对应的主要事件：onclick

radio单选按钮

对应的主要事件有onclick

文档对象模型DOM

HTML DOM定义了访问和操作HTML文档的标准方法。

  • D（文档）：当创建一个网页并将其加载到Web浏览器中时，DOM便根据这个网页创建出一个文档对象
  • O（对象）：一种独立的数据集合
  • M（模型）：文档对象的树状模型，元素与内容表现为一个个相互连接的结点（node）
    DOM中的结点类型包括：
    1.元素结点
    2.文本结点
    3.属性结点
