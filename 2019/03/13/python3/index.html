python数据分析(1):内建序列函数

2019-03-13 学习教程

文章目录

  1. enumerate
  2. sorted
  3. zip
  4. reversed

Python有很多内建序列函数

enumerate

Python内建了enumerate函数，返回（i，value）元组的序列，其中i是索引，value是元素的值

sorted

sorted函数返回一个根据任意序列中的元素新建的已知排序列表。

zip

  • 将列表，元组或其他序列的元素配对时，新建元组构成的列表
  • zip可以处理任意长度的序列，它生成列表长度由最短的序列决定
  • 有时候会enumerate同时使用
  • zip函数有一种方式去拆分序列。

reversed

reversed函数将序列的元素倒序排列
